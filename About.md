# Über TUDa HiveMind

TUDa HiveMind ist eine kollaborative Wissensdatenbank für Informatik-Studierende an der TU Darmstadt.

## Was ist das?

Eine durchsuchbare, verlinkte Sammlung von Vorlesungsnotizen, Zusammenfassungen und Lernmaterialien für den B.Sc. Informatik. Die Inhalte werden von Studierenden für Studierende erstellt und gepflegt.

## Mitmachen

Beiträge sind willkommen! Die Inhalte werden im separaten Content-Repository gepflegt:

**Content Repository:** [github.com/Juliper/TuDa_HiveMind](https://github.com/Juliper/TuDa_HiveMind)

### So kannst du beitragen:
1. Fork das Content-Repository
2. Erstelle oder bearbeite Markdown-Dateien
3. Erstelle einen Pull Request
